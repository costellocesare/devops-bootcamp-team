Examen Git y Github

Alumn@: [JOHANA MARTINEZ]

  1. Que es un sistema de control de versiones?

    [Es un sistema que permite trabajar en equipo en simultáneo sobre un mismo proyecto, efectuando diferentes cambios, en diferentes momentos del proyecto y controlar cada cambio que se haga ]

  2. Cual es la diferencia entre git y github?

    [git y git hub son diferentes, ya que git permite el manejo del versionado mientras que github permite tener alojados los repositorios]

  3. Cómo creo una rama en git?

    [para crear una rama en git se digité el comando git checkout -b + nombre de la rama o git branch 6 nombre de la rama]

  4. Cómo clono un repositorio?

    [desde git hub copiamos el código HTTPS y lo pegamos en la consola de git bash con el comando git clone + código git hub]
